Open Computing Facility Apt Repository

This apt repo contains packages used by the Open Computing Facility which aren't available from the main archives. Almost all packages are either unofficial backports or patches to existing Debian packages.

This repository also provides source packages for most of the binary packages here. If you need the source to any other binary package in this repo, please email us at help@ocf.berkeley.edu and we will do our best to point you to the source code or provide you the source.

This repository is unlikely to be useful to you. Perhaps you're looking for our mirrors of the official archives?
