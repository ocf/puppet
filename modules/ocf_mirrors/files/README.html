The Open Computing Facility at the University of California, Berkeley hosts mirrors of free and open-source software project archives.

These mirrors are available via:

  • http: http://mirrors.ocf.berkeley.edu/
  • https: https://mirrors.ocf.berkeley.edu/
  • rsync: rsync://mirrors.ocf.berkeley.edu/

To request for another project to be mirrored or to report a problem, send us an email.

If you find this mirror helpful, please donate to the OCF. Thank you.
